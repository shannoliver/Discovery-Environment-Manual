TestData folder contents

A quick description of each of the sample data files provided in the iPlant Discovery Environment.

accepted_hits_10k.sam

This SAM file provides G2P test data.

acer-traits.csv

This file contains csv trait data.

acer-tree.txt

This is a Newick file.

acer.nex

This is a NEXUS file with both tree and trait data.

aesculus-traits.csv

This file contains csv trait data.

aesculus-tree.txt

This is a Newick file.

aq.trait.nex

This is a NEXUS file containing trait data.

aq.tree.nex

This is a Newick file.

aquilegia-traits.txt

This file contains csv trait data.

aquilegia-tree.txt

This is a Newick file.

aquilegia.nex

This is a NEXUS file with tree data.

arcostap.nex

This is a NEXUS file with tree and trait data.

arctostaphylos-traits.csv

This file contains csv trait data.

arctostaphylos-tree.txt

This is a Newick file.

CsetH1.csv

This file contains csv trait data.

CsetH1.phy

This is a Newick file.

DsetH1.csv

This file contains csv trait data.

DsetH1.phy

This is a Newick file.

landsberg.v9_10k.sam

This SAM file provides G2P test data.

lobelioids-traits-disc.csv

This is one of a set of csv trait files.

lobelioids-traits-disc2.csv

This is one of a set of csv trait files.

lobelioids-traits-disc3.csv

This is one of a set of csv trait files.

Lobelioids-tree.txt

This is a Newick file.

PDAP.tree.nex

This file contains a phylogenetic tree.

PDAP.trait.nex

This is a Nexus file with trait data.

s_1_sequence_10k.txt

This file provides G2P test data.

s_8_sequence.clipper.sanger.txt

This file provides G2P test data.

s_8_sequence_10k.txt

This file provides G2P test data.

shorebirds.trait.nex

This file represents a set of continuous traits for the 70 bird species supported in the tree file shorebirds.tree.nex.

shorebirds.tree.tex

This file represents a phylogenetic tree for 70 species of birds.

SRR026741.zmv2_10k.sam

This SAM file provides G2P test data.

SRR026996.zmv2_10k.sam

This SAM file provides G2P test data.

SRR039512_10k.fastq

This FASTQ file provides G2P test data.

SRR039514_10k

This FASTQ file provides G2P test data.
