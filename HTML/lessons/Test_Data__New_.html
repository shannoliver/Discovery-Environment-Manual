Test Data (New)

A quick description of each of the sample data files provided in the iPlant Discovery Environment. Data is organized based on the tool or analysis for which each is intended.

FASTX

•    s_1_sequence.txt - This data set is comprised of a single lane of 8,296,093 36bp Illumina 1.2 single-end sequence reads. This library is a composite of three barcoded sequence libraries: 1) Barcode3: CTCGT; 2) Barcode6: CGACT; and 3) Barcode9: TAGCT.

•    s_5_1_sequence.txt - This data set is comprised of a single lane of Illumina 1.3+ reads, sequenced in the pair-end context. This file is run 1 and s_5_2_sequence.txt is run 2. This library is a composite of three barcoded sequence libraries: 1) Barcode1: GATCT; 2) Barcode2: ATCGT; and 3) Barcode 12: ACAGT. There are many sequences in this data that are of low quality, so the user will wish to filter them out before proceeding. After filtering, the user will wish to reconcile the reads, such that only reads where both Run1 and Run2 records exist are propagated into downstream analysis. Finally, the user will wish to convert these sequence reads from PHRED64 (Illumina 1.3) format to PHRED33 (Sanger) quality scaling.

•    s_5_2_sequence.txt - This data set is comprised of a single lane of Illumina 1.3+ reads, sequenced in the pair-end context. This file is run 2 and s_5_1_sequence.txt is run 1. This library is a composite of three barcoded sequence libraries: 1) Barcode1: GATCT; 2) Barcode2: ATCGT; and 3) Barcode 12: ACAGT. There are many sequences in this data that are of low quality, so the user will wish to filter them out before proceeding. After filtering, the user will wish to reconcile the reads, such that only reads where both Run1 and Run2 records exist are propagated into downstream analysis. Finally, the user will wish to convert these sequence reads from PHRED64 (Illumina 1.3) format to PHRED33 (Sanger) quality scaling.

•    s_8_sequence.txt -This is a dataset comprised of 6632564 100 bp reads from Arabidopsis thaliana that were generated using a protocol that may result in a terminal 3' sequence adapter and is useful to learn and test our QC preprocessing (FASTX toolkit). (CTGTAGGCACCATCAATCGTATGCCGTCTTCTGCTTG)

TE

CACE
•    aesculus-traits-T.csv
•    aesculus-traits.csv – This file is the trait selected for analysis after the tree has been selected, the data is uploaded to the DE and the parameters are set.
•    aesculus-tree.txt – This file is the tree selected for analysis after data is uploaded to the DE and the parameters are set.
•    Results
    •    In.LeafArea.cm2_nodes.txt – This file contains ancestral character estimates.
    •    In.LeafArea.cm2_out.txt – This file is an ancestral character reconstruction as an estimate of the Brownian motion parameter and the log likelihood of the model.
    •    In.LeafArea.cm2_plot.pdf – This is a visual plotting of the ancestral trait value estimates on the phylogenetic tree using a color gradient.
    •    In.LeafArea.cm2_tree.txt – This file is a table of ancestral trait values and the corresponding 95% confidence intervals.

Contrast
•    arctostaphylos-traits_2.txt

•    Results
    •    Outfile – File output by Contrast from reading a tree file that produces covariances, regressions, and correlations between characters for those contrasts.

DACE
•    aquilegia-traits_T.csv –
•    aquilegia-traits.csv – This file contains the trait data that have been matched to a tree’s data.
•    aquilegia-tree.txt – This file is the tree data that must have matching trait data
•    Results.
    •    T1_nodes.txt – This file is an estimation of the transition rates and the proportional likelihoods of the ancestor’s states.
    •    T1_out.txt – This file the proportional likelihood of all possible states at the internal nodes.
    •    Ti_plot.pdf – This file is a visual plotting on the phylogenetic tree using pie charts to represent the likelihoods for all possible states at the internal nodes
    •    T1_tree.txt – This file is an estimate of the transition rate with its associated uncertainty and the log likelihood of the model.
